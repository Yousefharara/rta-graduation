# Relief Track Aid (RTA)

Relief Track Aid (RTA) is a humanitarian aid distribution platform designed to improve transparency, fairness, and efficiency in aid delivery operations within the Gaza Strip.  
The system helps administrators and local organizations manage beneficiaries, monitor aid distribution processes, and coordinate humanitarian assistance effectively.

---

## Project Overview

The platform aims to solve common challenges in humanitarian aid distribution such as:

- Lack of transparency
- Duplicate beneficiary records
- Unorganized aid distribution
- Difficulty tracking aid delivery
- Internet connectivity interruptions during field operations

The system provides a centralized platform for managing aid operations while supporting offline functionality and beneficiary verification.

---

## Main Features

### Admin Features
- Manage local organization accounts
- Manage beneficiaries and user accounts
- Monitor aid distribution operations

### Local Organization Features
- Add and manage beneficiaries
- Verify beneficiary eligibility
- Manage aid distribution operations
- Work in offline mode with synchronization support

### Beneficiary Features
- Login using generated credentials
- Track aid distribution status
- View previously received aid batches

### Donor Features
- Donate through the platform without account registration
- View campaign information

---

## System Architecture

The system follows a client-server architecture consisting of:

- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MySQL
- Authentication & Authorization System
- RESTful APIs
- Offline synchronization support
